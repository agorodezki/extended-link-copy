Extended Link Copy Settings

Plugin for Firefox and Chrome to copy parts of links to the clipboard.

  • Title: Choose a title for your entry
  • Regex: A regex that describes the part of a href in Javascript notation. See here.
  • Compose: If you have set capturing groups in your regex you can access them with ${n} in this field. The n stands for a number greater 0. With ${0} you get the hole match of the regex. With ${1} you get the content of the first capturing group, with ${2} the content of the second, and so on. Everything else you're writing to this field would be assumed to the resulting string. If you leave this field empty the hole match of the regex would be assumed to the resulting string. See examples below or on Github. If you can't see examples below click on 'reset to factory defaults'.

Every changes made to the settings have to be saved, even deleted items. If you made changes that shouldn't be saved or accidentally clicked the reset button just reload the settings page.

Options in the settings list can be accessed under its title in the context menu of every hyperlink. If you have entered more then one option, they're grouped under 'Extended Link Copy'.
